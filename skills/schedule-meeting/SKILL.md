# Schedule Meeting

Skill для согласования встречи с пользователем.

## Алгоритм

1. **Инициация** (owner запрашивает)
   - Определить участника (@username или имя)
   - Получить временные слоты владельца
   - Создать задачу согласования в БД

2. **Согласование** (через сессию участника)
   - Отправить запрос участнику через его сессию
   - Дождаться ответа с удобным временем
   - При коллизии — уведомить владельца, повторить

3. **Подтверждение**
   - Согласовать финальное время
   - Уведомить всех участников
   - Создать событие в scheduler (напоминание за 5 мин)

4. **Напоминание**
   - За 5 минут до встречи создать Яндекс Телемост
   - Отправить ссылку всем участникам

## Использование

```
/schedule-meeting @username завтра 12:00-20:00
```
